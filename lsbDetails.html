Audio Steganography Using LSB

Audio Steganography using LSB (Least Significant Bit) is a data hiding technique that secretly embeds text or file data inside an audio file without any audible change to the sound. The hidden information is completely invisible to anyone listening to the audio, making it a powerful tool for secure and covert data transmission.

This project is implemented in Python using NumPy and SciPy libraries. The system achieved an SNR (Signal-to-Noise Ratio) of 58.42 dB, which confirms that the audio quality is excellently maintained even after hiding data. The extracted data matches the original secret data perfectly.

Working Principle

  • Load the original audio WAV file using SciPy
  • Convert secret message to binary (8 bits per character)
  • Replace the Least Significant Bit of each audio sample with one data bit
  • The change per sample is only 1 bit — completely inaudible
  • Save the modified audio as a Stego Audio file
  • To extract: read LSBs of each sample and reconstruct the message

Main Components / Tools Used

  • Python 3.x
  • NumPy — array and bit manipulation
  • SciPy — WAV audio file reading and writing
  • WAV audio files (16-bit PCM format)
  • Termination sequence (11111111) to mark end of data

How LSB Works

  • Original sample (16-bit): 0010110011010110
  • Last bit (LSB) changed to embed one data bit
  • After embedding: 0010110011010111
  • Change is only 1 out of 65536 levels — completely inaudible

Results Achieved

  • SNR: 58.42 dB — Excellent audio quality maintained
  • Extracted data matches secret data 100% accurately
  • Supports embedding text, file content, and binary data

Features

  • Completely inaudible changes to the audio file
  • Supports text or file hiding inside WAV audio
  • Secure — hidden data is invisible without the extraction key
  • High SNR ensures excellent audio quality after embedding
  • Fast processing using NumPy array operations

Applications

  • Covert secure communication
  • Digital watermarking of audio content
  • Copyright protection
  • Cybersecurity and data hiding research
