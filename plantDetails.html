Automatic Plant Watering System

The Automatic Plant Watering System is a smart irrigation solution that monitors soil moisture levels and automatically waters plants when the soil becomes dry. It eliminates the need for manual watering and ensures plants always receive the right amount of water at the right time.

A soil moisture sensor is inserted into the soil and continuously reads the moisture content. When the moisture drops below a set threshold, the Arduino activates the relay module which switches on the water pump. The system also displays the current moisture percentage and pump status on a 16x2 LCD screen.

Working Principle

  • Soil moisture sensor measures the water content in the soil
  • Arduino reads analog value from sensor pin A0
  • If moisture is below threshold — relay activates and pump turns ON
  • If moisture is sufficient — relay deactivates and pump turns OFF
  • LCD displays live moisture percentage and pump status
  • Process repeats continuously every second

Main Components Used

  • Arduino UNO
  • Soil Moisture Sensor (Analog)
  • Relay Module — 5V
  • Mini Water Pump — 5V DC
  • LCD Display 16x2 with I2C module
  • Water pipe / tube
  • Power Supply 5V
  • Jumper wires and breadboard

Pin Connections

  • Moisture Sensor OUT → Arduino A0
  • Relay IN → Arduino D8
  • VCC → 5V | GND → GND
  • LCD SDA → A4 | LCD SCL → A5

Features

  • Fully automated — no manual intervention needed
  • Saves water by watering only when required
  • Real-time moisture display on LCD
  • Low-cost and easy to build
  • Suitable for indoor and outdoor use

Applications

  • Home gardens and indoor plants
  • Greenhouses and nurseries
  • Agriculture fields (scaled up)
  • Terrace and balcony farming

Future Improvements

  • Add IoT (ESP8266) for remote monitoring via mobile app
  • Add multiple sensors for large garden areas
  • Integrate solar panel for outdoor power
  • Add rain sensor to avoid watering during rainfall
